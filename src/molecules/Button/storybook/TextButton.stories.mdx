## Usage

Using the Text Button is as almost as easy as the Contained Button. The only variation is passing the `variant` prop
with the argument of `text`.

```jsx
<Button variant="text">This is a text button</Button>
```

#### Adding Icons

You can pass any ReactNode into the endIcon or startIcon props to be rendered at the end or beginning of your Button.

```jsx
<Button variant="text" endIcon={<Coffee />}>
  Coffee Time!
</Button>
```
