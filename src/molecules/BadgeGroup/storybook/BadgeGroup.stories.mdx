# Badge Group

The `BadgeGroup` component is a wrapper for the `Badge` component. It is often used to convey a message to users to
assist with UX or to notify the user with important information.

## Usage

Its a wrapper! Using the `BadgeGroup` is a easy as wrapping a badge and giving it a title.

```jsx
<BadgeGroup title="I am in the big badge">
  <Badge>I'm in the little badge</Badge>
</BadgeGroup>
```

The `Badge` and `BadgeGroup` have full use of CSS quarks for styling of `textColor` and/or `backgroundColor`.

### Position

By default the `Badge` will be at the start of the BadgeGroup. If you need the badge to be at the end of group you can
make use of the `badgePosition` prop and simply set it to `end`

```jsx
<BadgeGroup title="I am in the big badge" badgePosition="end">
  <Badge>I will appear to the right of the title</Badge>
</BadgeGroup>
```

### Icons

Set an icon with the `icon` prop. Icons can be any [React Icon](https://react-icons.github.io/react-icons/) or SVG. **If
using and SVG, you have to set the `width` on the SVG**
