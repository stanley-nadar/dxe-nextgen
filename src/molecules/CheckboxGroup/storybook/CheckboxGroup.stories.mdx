# Checkbox Groups

Checkbox groups are a great way to make checkboxes more interesting, or to add more information to the user's options
without adding clutter. When options are explicitly separated into cards, it's obvious which elements belong to which
group and easier to quickly distinguish between checkbox options.

## Usage

#### Label

The `label` prop is required and inherently light/dark mode optimized

#### Checkbox size

The `checkboxSize` prop is optional and determines the size of the `Radio` or `Checkbox`

#### Control Style

The `controlStyle` prop is optional and determines if the `CheckboxGroup` displays a `Radio` or `Checkbox`

#### Icon

The `icon` prop is optional and displays an icon on the left side of a `CheckboxGroup`. If an icon exists, the `Radio`
or `Checkbox` is displayed on the right
