# Hero Form Component

The Hero is the first thing users see when arriving on a Freshworks website. This variant prioritizes the usage of a
form.

The storybook has a few words in camel case format due to a translation package. This will not appear in the live build.

### Media reference

The Content Editor can reference a static image, the `Video Component`, `Accolades Component`, or the
`Testimonial Component.`

### Background

For Mobile and Tablet, the background covers the entire section. Where Desktop covers The Copy and Media Reference.
Content editors select a background from the dropdown menu.

### Hero Feature List

References the Component: List, which takes in references of List Items with proper markup.

Inside List Items, the content editor can select whether he or she wants a symbol next to the list title.
