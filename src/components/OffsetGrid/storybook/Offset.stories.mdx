# Offset Grid Component

The Offset Grid Component will be used to link to Freshworks pages. It contains a `eyebrow`, `heading`, a set of cards,
and an image.

Each Offset Card will contain a Heading, a brief description, and a link to its corresponding page. The heading has a
max character limit of 45 and the description has a max character limit of 256.
