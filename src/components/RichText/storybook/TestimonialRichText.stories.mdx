# Testimonial Rich Text Component

This is the Rich Text Version of the `Testimonials Component`. Utilize the Testimonial Card Content Model when inputting
this into Rich Text.

The width of the Component is dependent upon the parent element's width. See the Sign-Up-Form as a reference.

## Person

Comes from the Content Model: Entity Person, the full name and role field are required for full functionality

## Quote

Uses Rich Text, this allows for the user to bolden, italicize or hyperlink certain excerpts.

## Image

The image is forced to be set to have a height and width of 80px.
