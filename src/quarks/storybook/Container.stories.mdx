# Container

The Container is a simple `<div>` element. It accepts all basic props and nothing else, making it a good representation
of what styling props exist on all quarks.

### `As` Prop

The quarks are built using `styled-components`, and include all the features of them including the `as` prop, which
allows you to change the element tag by passing in a string value corresponding with the target tag type.

Ex: Pass in `"table"` to render the quark as a `<table>` element while still benefitting from all styling props that
exist on Container.
